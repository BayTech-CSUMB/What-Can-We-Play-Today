1 · User Library GetOwnedGames · no rate limit

Fast lookup — returns the full list of games a user owns. No appdetails enrichment here, so no rate limiting applies.

2 · Single Game Details appdetails · rate limited

Full enriched data for one game — name, price with discount, all Steam categories, genres, description. Uses the 1500ms rate-limited queue.

3 · My Multiplayer Games streaming · sorted by playtime

Games stream in one-by-one as they're enriched — sorted by playtime so your most-played games appear first. Cached results appear instantly; new lookups arrive every ~1.5s.

4 · Shared Multiplayer Games core feature

The main WhatCanWePlay feature — finds multiplayer games owned by all listed users. Filter the results by category, genre, or price.
